Ini adalah project mini couse revou 7 June 2023

Saya suka menulis coding dalam bahasa Javascript(React Native), Nodejs, dan Java karena kebutuhan pada React Native.

Mula-mula saya tertarik dalam programing language bermula karena saya kesal dalam bermain game Point Blank karena terlalu sering kalah, Makanya sebab itu saya belajar mencari solusi bagaimana saya bisa membunuh musuh tanpa harus ribet (Cheat), namun pada tahun awal-awal saya belajar antara tahun 2014/2015 sangat sulit.

tapi saya tidak patah semangat, dan masa itu saya menemukan salah 1 orang (Juwendi) bekerja di zepetto, di situlah titik terang saya belajar programing. dia banyak mengajarkan saya mulai dari mencari bytecode mengguna cheat-engine untuk di manipulasi, dan merekayasa ulang dalam bahasa c++, membuat injector menggunakan vb6, melakukan bypass pada anti virus. dan banyak lagi yang saya pelajari dari beliau. tapi pada awal-awal tahun 2015 akhir saya bosan dengan game Point Blank.

dan mencari sasaran game lain dan pada saat itu saya menemukan game Crazy Shooter dari gemscool, saya dan beberapa kawan saya mendirikan komunitas HackEngine dalam meretas game untuk di jual (Cheat) berdurasi. dan pada tahun-tahun itu juga saya membuat aplikasi untuk membaypass wifi.id, karena pada masa-masa itu internet paling cepat di daerah saya hanya wifi.id. namun karena berbayar makanya saya membypassnya. mungkin jika kalian mencari di google dengan keyword khalis afkari wifi.id semoga masih ada :D atau kalian bisa mencari cheat CrazyShooter di youtube 8 tahun lalu

namun setelah game CrazyShooter tutup kami membubarkan komunitas HackEngine karena sudah tidak tertarik lagi, dan juga beberapa kawan sudah mulai berumah tangga. dan pasa masa itu saya mulai menikmati kembali hidup saya yang dulunya saya sering di depan komputer sekarang saya sudah lebih banyak bertemu sama orang-orang di lingkaran saya.

maaf saya bercerita panjang lebar, namun itu kisah saya, jika tidak suka membaca kisah saya, maka bisa di abaikan, jika cara saya menulis terbelit-belit, saya hanya bisa mengatakan saya bukan penulis, saya hanya menceritakan kisah saya.

Mungkin masih banyak yang bisa di ceritakan, tapi seperti saya juga malas menceritakannya, mungki cukup disini saja pengalaman singkat saya.
